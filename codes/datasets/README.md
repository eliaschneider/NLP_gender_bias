# List of datasets used in this repository
Additional dataset available here: http://hatespeechdata.com/ 


## Italian datasets 
1. HaSpeeDe - EVALITA 2018 contest on Hate Speech detection
http://www.di.unito.it/~tutreeb/haspeede-evalita18/index.html

2. HSC - Italian Twitter Corpus of Hate Speech against immigrants 
https://github.com/msang/hate-speech-corpus, used twitter API 
to translate tweet ids in text


## English datasets 
3. StormfrontWS - Sentence from White Supremacist Forum annoted 
https://github.com/aitor-garcia-p/hate-speech-dataset

4. Davidson et al. - Hate speech and offense in Tweets 
https://github.com/t-davidson/hate-speech-and-offensive-language
